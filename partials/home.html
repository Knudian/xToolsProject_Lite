Welcome on the xTools Project LITE

This project is made of tools for the MMORTS game Travian : Kingdoms .

The Genesis of the the project was done by some french players, in order to have a more complete tool to help the players, and not to have many external tools scattered all over the web.

This Lite version of the project include only the stand-alone tools. Those tools are the most used ones, and publishing those here, on GitHub, will make it easier for all developpers to improve an existing project.

I hope you will enjoy all those tools, and help improve them.

Tool list :
Hammer Builder :
Tool for estimating the troops amounts in a hammer-type offensive army.
TSA :
Tool for estimating the Tournament Square (TS) of an ennemy.
Troop Research :
Tool defining the minimum troop amount for having efficient troop upgrades.
